   genpgm program rev 2.0 2022
USAGE:
  genpgm <prog_name> <#in files> <#out files>
     creates a cc program <prog_name> with input
     and output files.

 eg.  genpgm floopy 1 2
        -- generates the program floopy.cc with one input file
           and two output files

        -- generates makefile
