 gencc program rev 1.2 2016
USAGE:
  gencc <ifile_name> <ofile_name> 
EXAMPLE:
gencc x1.vhd genx1.cc 
 reads x1.vhd, produces genx1.cc, makefile
 where makefile is for gcc on linux
 The resulting genx1 program will reproduce x1.vhd.
